<H2 id="start">📖 How to Start</H2>

```bash
# Clone this repository
$ git clone https://github.com/iwataniKenji/data-structure-page.git

# Access the project directory
$ cd data-structure-page

# Install dependencies
$ yarn

# Start application
$ yarn start

```
